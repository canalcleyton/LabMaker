Instalação do Python e Bibliotecas

Aqui está um roteiro passo a passo para instalar o Python e as bibliotecas necessárias (pyserial e paho-mqtt) em um sistema Windows:

1. Instalar o Python

  1. Baixar o Instalador:

    • Acesse python.org.
    • Clique em "Download Python" (certifique-se de baixar a versão mais recente).

  2. Executar o Instalador:

    • Abra o arquivo .exe baixado.
    • Marque a opção "Add Python to PATH" no início da instalação.
    • Clique em "Install Now" e siga as instruções na tela.

  3. Verificar a Instalação:

    • Abra o Prompt de Comando (cmd).
    • Digite o seguinte comando para verificar a instalação: 
      python --version

2. Instalar as Bibliotecas Necessárias

  1. Abrir o Prompt de Comando:

    • Pressione Win + R, digite cmd, e pressione Enter.

  2. Instalar pyserial:

    • No Prompt de Comando, digite: 
      pip install pyserial

  3. Instalar paho-mqtt:

    • Após instalar o pyserial, digite: 
      pip install paho-mqtt

3. Verificar as Instalações das Bibliotecas

Para verificar se as bibliotecas foram instaladas corretamente, você pode abrir um interpretador Python:

  1. No Prompt de Comando, digite:

              python

  2. Tente importar as bibliotecas:

              import serial
import paho.mqtt.client as mqtt

Se não houver mensagens de erro, as bibliotecas foram instaladas com sucesso.

4. Criar um Script de Teste (Opcional)

Para garantir que tudo está funcionando, você pode criar um simples script Python:

  1. Abra um editor de texto e cole o seguinte código:

              import serial
import paho.mqtt.client as mqtt
import time

print("Bibliotecas importadas com sucesso!")

  2. Salve o arquivo como test_script.py.

  3. No Prompt de Comando, navegue até o diretório onde você salvou o arquivo e execute:

              python test_script.py

Se você vir a mensagem "Bibliotecas importadas com sucesso!", tudo está funcionando corretamente!
